# AVR-labs

## Doel

Deze repository heeft als doel het leren over microcontrollers toegankelijk te maken voor iedereen, ook zonder fysieke hardware.  
Studenten kunnen ontdekken hoe een echte microcontroller werkt met behulp van simulatie, nog vÃ³Ã³r ze met een echt bord aan de slag gaan.  

Arduino wordt gebruikt als een nuttig hulpmiddel, maar niet als een zwarte doos die verbergt hoe dingen werkelijk werken.  
Elk programma wordt opgebouwd als echte firmware, precies zoals die zou draaien op een ATmega328P.  

Dezelfde code kan zowel gesimuleerd als naar flashgeheugen geschreven worden, wat aantoont dat software niet afhankelijk is van een specifiek bord.  
Door simulatie, flashen en labs van elkaar te scheiden, leren studenten hoe embedded systemen in de praktijk zijn opgebouwd.  

Het buildsysteem is ontworpen om mee te groeien van eenvoudige oefeningen naar grotere projecten.  
Simulatie maakt leren mogelijk in omgevingen waar hardware niet beschikbaar of te duur is.  

Elk lab legt de nadruk op begrip, niet alleen op het laten werken van iets.  
Het doel is niet alleen microcontrollers gebruiken, maar ze echt begrijpen.
